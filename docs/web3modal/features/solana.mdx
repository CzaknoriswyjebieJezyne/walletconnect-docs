# Web3Modal Solana

The Web3Modal SDK supports Solana, allowing users to connect their Solana wallets to applications. Web3Modal provides a simple, secure, and seamless in-app experience for users looking to transact within web3.

:::caution

Web3Modal Solana is in canary and is not recommended for production use.
:::
